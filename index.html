Intro

My experience in backend development is complemented by a solid foundation in HTML, CSS (including Bootstrap, Less, and Tailwind), and JavaScript frameworks like Next.js and Vue. System design is a particular area of interest for me, and I always strive to create architectures that are both scalable and resilient.
